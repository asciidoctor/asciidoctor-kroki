= Asciidoctor Kroki Extension
:description: An Asciidoctor extension to convert diagrams to images using Kroki.

Asciidoctor Kroki is an extension for https://github.com/asciidoctor/asciidoctor.js[Asciidoctor.js] and https://github.com/asciidoctor/asciidoctor[Asciidoctor] to convert diagrams to images using https://kroki.io[Kroki].

Kroki provides a unified API with support for many diagram libraries.
Write your diagram source directly in AsciiDoc, and Kroki renders it as an image.

== Supported environments

The extension is available for:

* *Node.js* — use it in your build scripts or Asciidoctor.js-based toolchains
* *Browser* — use it directly in the browser
* *Ruby* — use it with the Asciidoctor Ruby gem
* *Antora* — integrate it into your Antora documentation site
